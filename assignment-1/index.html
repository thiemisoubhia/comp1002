The HTML Pizza Company

For this project, I created a fully functional HTML page that allows users to order one or more pizzas. The page features a variety of form inputs and controls designed to collect all the necessary details for a pizza order, making the experience interactive and user-friendly.

Project Highlights

Structured HTML: The page is built with proper HTML5 semantics, including header, main, and footer elements to create a clear page outline.

User-Friendly Form: The form collects essential information such as: Number of pizzas Size and shape of the pizza Choice of toppings Crust type (deep dish, thin crust, etc.) Dining preference (takeout, delivery, or eat-in)

Variety of Inputs: The form includes text boxes, radio buttons, checkboxes, select menus, text area, and submit buttom, ensuring a complete and intuitive ordering experience. Visual Elements: Includes at least one relevant image with an alt attribute to enhance the page visually.

Best Practices: The code contains helpful HTML comments, proper formatting, and regular commits documenting progress.
